## Troubleshooting Tips for Phase 2

### Common Issues

**Issue**: `uniffi-bindgen not found`
```bash
Solution: cargo install uniffi_bindgen
```

**Issue**: Cannot find wallet.udl
```bash
Solution: Verify path in build.gradle.kts is: ../wallet.udl (not absolute path)
```

**Issue**: Type mismatch between Rust and Kotlin
```bash
Solution: Check wallet.udl syntax - Dictionary must match Rust struct fields
```

**Issue**: JVM compilation fails
```bash
Solution: 
1. Run: gradle generateUniFFI (before compiling)
2. Check Kotlin version >= 1.9.0
3. Verify JDK 11+ installed
```

**Issue**: Tests failing
```bash
Solution:
1. Ensure Rust library compiled successfully (Phase 1)
2. Check native library (.so or .dll) is in path
3. Verify test setup matches Kotlin conventions
```
